# 1. 자바 개념 저리

## (1) 자바란?
	- 실생활을 모델링하여 객체로 구현하는 프로그래밍 언어
	- 객체지향 언어(Object Oriented Languge) --> 객체지향 프로그래밍(OOProgramming)
	- 제임스 고슬링 : IBM : IBM Green Project > Oak > Java
